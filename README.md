# MyExpenseTracker

MyExpenseTracker is a React Native expense tracker with offline storage, quick-add system bubble, and monthly insights with charts.

## Features

- Quick transaction capture with categories and accounts
- Offline SQLite storage and local analytics
- Monthly insights with daily income/expense line graph
- Country-based currency selection
- System floating bubble for fast entry

## Getting Started

### 1) Install dependencies

```sh
npm install
```

### 2) Start Metro

```sh
npm start
```

### 3) Run on Android

```sh
npm run android
```

### 4) Run on iOS (optional)

```sh
bundle install
bundle exec pod install --project-directory=ios
npm run ios
```

## Notes

- The home screen shows current-month totals and transactions.
- Insights includes month selection and category breakdowns.
- Settings allows choosing currency and toggling the system bubble.
